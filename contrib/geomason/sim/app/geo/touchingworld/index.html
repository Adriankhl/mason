Touching World
This demo shows how to compute geometry that touches other geometry. A set of arbitrary polygons is displayed. Unselected polygons are colored blue; one is selected, which is indicated by being red. With each time step the next selected polygon is chosen from those that are touching the currently selected.

As the simulation runs it reports to the console the currently selected polygon, those that are touching it, and which one was randomly selected. A sample is shown below:
